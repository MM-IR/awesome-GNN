## 1.创建图形

```
import networkx as nx
G = nx.Graph() #这个就是有一组节点和已识别的节点对的集合。在networkx中，节点可以是任何的hashable对象，比如字符串/图像/XML对象等等。

也可以是另一个图形或者自定义节点对象等。
```

## 2。结点
```
1.添加一个节点:
G.add_node(1)

1.1.或者从任何iterable容器，比如列表中添加元素:
G.add_nodes_from([2,3])

1.2. 还可以添加节点和节点属性等等。

G.add_nodes_from([
     (4, {"color": "red"}),
     (5, {"color": "green"}),
 ])

这个就是节点属性color。

2.一个图中的节点还可以合并到另一个图中。（节点直接融进来）

H = nx.path_graph(10) # (0,1,2,3,...,9)
G.add_nodes_from(H)

-----

当然我也可以使用图表H作为一个节点G.



这个就是节点属性color。

2.一个图中的节点还可以合并到另一个图中。（节点直接融进来）

H = nx.path_graph(10) # (0,1,2,3,...,9)
G.add_nodes_from(H)

-----

当然我也可以使用图表H作为一个节点G.



这个就是节点属性color。

2.一个图中的节点还可以合并到另一个图中。（节点直接融进来）

H = nx.path_graph(10) # (0,1,2,3,...,9)
G.add_nodes_from(H)

-----

当然我也可以使用图表H作为一个节点G.



这个就是节点属性color。

2.一个图中的节点还可以合并到另一个图中。（节点直接融进来）

H = nx.path_graph(10) # (0,1,2,3,...,9)
G.add_nodes_from(H)

-----

当然我也可以使用图表H作为一个节点G.

3.边缘

1）G也可以通过一次添加一个边来生长。

G.add_edge(1,2)
e = (2,3)
G.add_edge(*e) #这里就是直接添加边2，3

2）或者添加边的列表
G.add_edges_from([(1,2),(1,3)])

3)也可以添加带权重的边
e = (2,3,{{'weight': 3.1415}})

4)添加现有的图的边
G.add_edges_from(H.edges)

5)删除所有节点和边
G.clear()

6）我们添加新的节点或者边缘的时候，Networkx会悄悄地忽略已经存在的任何节点/边缘。

G.add_node('spam') #这里就是增加一个叫spam的节点

G.add_nodes_from('spam') #这里就是增加4个节点，比如s/p/a/m。

7）我们还可以查看节点和边的数目。

G.number_of_nodes()
G.number_of_edges()
```
